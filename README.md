# FiskyCorp

We build secure, fast, and maintainable web products — from idea to production.

**FiskyCorp** is a small Sweden-based development team focused on shipping high‑quality websites and web applications with a strong emphasis on security, performance, and clean engineering practices.

---

## What we do
- **Web applications & SaaS** (Laravel, modern frontend tooling)
- **Websites** (marketing sites, landing pages, content sites)
- **Architecture & performance** (scalable foundations, speed optimizations)
- **Security-first development** (least privilege, reviews, CI, dependency hygiene)

---

## How we work
- **PR-first workflow**: changes go through pull requests with reviews
- **Automation**: CI checks and dependency updates (Dependabot) are standard
- **Documentation**: clear docs, templates, and consistent conventions
- **Professional repo hygiene**: CODEOWNERS, contributing guides, and security notes where appropriate

---

## Current project
### Tentahuben
A Sweden-focused platform for universities and student unions (föreningar) to securely upload, moderate, search, and access past exam PDFs. Built with Laravel.
